Introduction

Java is a high-level, object-oriented programming language widely used for cross-platform applications.

What you should know

Java programs are written inside of classes that contain methods. A simple print statement in Java looks like this:


public class Main {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

Key Points:

  • Java is statically typed.
  • All Java programs must be inside a class.
  • The `main` method is the entry point of a Java program.
  • Java is case-sensitive.
  • Statements end with a semicolon (`;`).

Data Types and Variables

Java variables must be declared with a specific type.


int year = 2025;
double pi = 3.14159265359;
float shortPi = 3.14f;
String text = "Java Is Cool";
boolean isJavaFun = true;
char grade = 'A';

We also have Shorts, and Longs as well!

Control Flow

Control flow determines the order in which statements execute.

Conditional Statements (if-else)

Condition statements allow a program to execute different code blocks based on a condition.


int num = 10;
if(num > 0) {
    System.out.println("Positive number");
} else {
    System.out.println("Negative number");
}

Loops

  • For Loop: Used for known iteration counts.
  • While Loop: Runs while a condition is true.
  • Do-While Loop: Ensures at least one execution.

Methods and Functions

Java uses methods to encapsulate reusable code blocks.


public class Calculator {
    public static int add(int a, int b) {
        return a + b;
    }

    public static void main(String[] args) {
        int sum = add(5, 10);
        System.out.println("Sum: " + sum);
    }
}
