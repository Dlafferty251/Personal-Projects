    • Introduction

    Java is a high-level, object-oriented programming language that is widely used for building cross-platform applications. It is known for its "Write Once, Run Anywhere" principle meaning. Compiled Java code run on any platform tht supports Java without the need for a recompile.

    Basic Syntax for Java

    Java Programs are written inside of classes that contain methods. A Simple print statement in Java looks like this:

    
                public class Main {
                    public static void main(String[] args) {
                        System.out.println("Hello, World!");
                    }
                }

    Explanation:

    public class Main defines a a class named Main

    Data types and Variables

    Java is statically typed, meaning variables must be declared with a specific type.

    
                int year = 2025;
                double pi = 3.14159265359;
                float pi = 3.14;
                String text = "Java Is Cool";
                boolean isJavaFun = true;
                char grade = 'A';

    We also have Shorts, and Longs as well!
